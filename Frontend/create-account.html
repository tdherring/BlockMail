Create Account
Pressing the "Generate Keys" button below will create three keys required to use BlockMail: Wallet Address (wallet.addr), Wallet Password (wallet.pass), and Mail Decryption Key (private.key). This may take from a few seconds to a minute; please be patient.
If you can't save them using the "Download" buttons, copy them into files and save them.

NOTE: If you lose these keys, it is IMPOSSIBLE to recover them, so keep them somewhere safe and have them backed up!

Purpose

BlockMail is a dissertation project developed by Tom Herring at Queen Mary University London in 2019. It serves as a proof of concept that Blockchain technology can be used as an email system for extremely secure, traceable communications.

More Information

If you would like to learn more about Blockchain technology, please go to Investopedia . This gives a good outline of the technology and its potential uses (both now and in the future). For any questions, please email t.d.herring@se17.qmul.ac.uk.
